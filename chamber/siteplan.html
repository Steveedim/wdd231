DiscoverNaija - Site Plan

Site Name

DiscoverNaija

DiscoverNaija is a website designed to showcase Nigeria Rich cultural heritage, historical landmarks, festivals and tourist attraction.

Optional domain availability: discovernaija.org

Site Purpose

The website will provide local and international travelers with curated travel guide, destination highlights and cultural insights. The website will have the following features

  1. Interactive homepage with popular destination and festival
  2. State by state Tourism directory
  3. Responsive design for both mobile and desktop view

Scenarios

  • Where to get Information concerning Nigerian culture and tourism?
  • Travel planning with DiscoverNaija?

Color Schema

  • Forest Green (#2E7D32): Used for headings and call-to-action buttons.
  • Ivory (#F9F7F1): Used as the primary background color for readability and warmth.

Typography

  • Montserrat: Used for main headings (clean, bold, modern).
  • Open Sans: Used for body text (easy to read, professional).
